# Creating a TypeScript Project

This guide walks through the basic setup of a TypeScript project using either **npm** or **Bun**.

---

# Step 1: Initialize the Project

Using **npm**:

```bash
npm init -y
```

Using **Bun**:

```bash
bun init -y
```

This creates a default `package.json` file, which stores your project's metadata, dependencies, and scripts.

---

# Step 2: Install TypeScript

Using **npm**:

```bash
npm install --save-dev typescript
```

Using **Bun**:

```bash
bun add --dev typescript
```

TypeScript is installed as a **development dependency** since it is only needed during development and compilation.

---

# Step 3: Generate the TypeScript Configuration File

Using **npm**:

```bash
npx tsc --init
```

Using **Bun**:

```bash
bunx tsc --init
```

This command generates a default **`tsconfig.json`** file, which controls how the TypeScript compiler behaves.

---

# Step 4: Create the Project Structure

Create the following folders:

```
project/
│
├── src/
├── dist/
├── package.json
└── tsconfig.json
```

### Folder Description

- **`src/`** – Contains all your TypeScript source code.
- **`dist/`** – Stores the compiled JavaScript output generated by the TypeScript compiler.

---

# Step 5: Write Your Code

Create a file named:

```
src/index.ts
```

Example:

```ts
console.log("Hello, TypeScript!");
```

---

# Step 6: Compile the Project

Using **npm**:

```bash
npx tsc
```

Using **Bun**:

```bash
bunx tsc
```

The TypeScript compiler reads your source files from the `src/` directory and generates the compiled output inside the `dist/` directory.

Depending on your `tsconfig.json` settings, the compiler may generate:

- **`.js`** – Executable JavaScript files.
- **`.js.map`** – Source maps used for debugging.
- **`.d.ts`** – Type declaration files (if declaration generation is enabled).

Example output:

```
dist/
├── index.js
├── index.js.map
└── index.d.ts
```

---

# Step 7: Execute the Compiled Code

Run the generated JavaScript file using Node.js:

```bash
node dist/index.js
```

> **Note:** TypeScript files (`.ts`) cannot be executed directly by Node.js. They must first be compiled into JavaScript.

---

# Step 8: Add Convenience Scripts

To avoid repeatedly typing long commands, define scripts in your `package.json` file.

```json
{
  "scripts": {
    "start": "node dist/index.js",
    "build": "npx tsc"
  }
}
```

Now you can use:

Compile the project:

```bash
npm run build
```

Run the compiled application:

```bash
npm start
```

---

# Running the Same Scripts with Bun

Bun uses the same scripts defined in `package.json`.

Compile the project:

```bash
bun run build
```

Run the application:

```bash
bun run start
```
